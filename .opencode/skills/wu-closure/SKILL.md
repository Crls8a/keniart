---
name: wu-closure
description: Close Keniart work units with evidence, risks, rollback, and follow-ups.
---

Use this structure when closing a work unit:

1. Summary
2. Files changed
3. Tests / Validation evidence
4. Tests / Validation not run, with reason and risk
5. Risks / open questions
6. Rollback suggestion

Do not report implementation details as validation. Validation must name the command or inspection performed and its result.
