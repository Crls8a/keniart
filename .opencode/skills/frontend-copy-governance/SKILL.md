description: Govern user-facing copy in Keniart frontend work; use when editing UI labels, content, or gallery text.
---

Rules:

- Keep user-facing gallery copy intentional and consistent with the existing product tone.
- Avoid scattering repeated UI copy across components.
- For reusable sections, prefer feature-local content constants when the project already has that pattern.
- Do not introduce source-project naming or unrelated product copy into Keniart.
