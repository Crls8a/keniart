# Obras pendientes de carga

Completar una seccion por obra real. Si un dato no esta confirmado, dejarlo como `PENDIENTE` en vez de inventarlo.

## Plantilla

```md
## Titulo de la obra

- slug sugerido: `titulo-de-la-obra`
- anio: PENDIENTE
- serie: PENDIENTE
- tecnica: PENDIENTE
- soporte: PENDIENTE
- medidas: alto x ancho x profundidad cm
- estado: available | reserved | sold | not_for_sale
- precio: PENDIENTE / privado / a consultar / monto + moneda
- descripcion corta: PENDIENTE
- tags: PENDIENTE
- seleccion dossier: si/no
- destacada en home: si/no
- imagen principal: `public/artworks/incoming/nombre-del-archivo.ext`
- imagenes detalle: PENDIENTE
- imagen en ambiente: PENDIENTE
- notas de carga: PENDIENTE
```

## Obras reales

Agregar aca las obras cuando lleguen las imagenes y fichas definitivas.
